# Курс «Разработчик C#/.NET»

Практический курс из 35 уроков: от устройства языка C# и ООП до конкурентности, архитектуры, интеграции и CI/CD. Материалы рассчитаны на самостоятельное чтение и работу с преподавателем через Git.

## Как устроен курс

Курс рассчитан на человека без технического образования и опыта программирования. Начните с [путеводителя новичка](МАТЕРИАЛЫ/ПУТЕВОДИТЕЛЬ_НОВИЧКА.md); встречая незнакомое слово, обращайтесь к [глоссарию](МАТЕРИАЛЫ/ГЛОССАРИЙ.md).

Каждый урок содержит три файла:

- `ТЕОРИЯ.md` — самостоятельный конспект, примеры, ограничения и типичные ошибки;
- `ВОПРОСЫ.md` — вопросы для самопроверки и обсуждения с преподавателем;
- `ПРАКТИКА.md` — задание, критерии готовности и дополнительные уровни сложности.

Теория читается последовательно. Не требуется заранее знать английские термины, устройство компьютера, Git или математику сверх школьного уровня. Термин сначала объясняется простыми словами и только затем используется в профессиональной форме.

Ученик создаёт solution и проекты прямо в каталоге урока. Исходники не должны заменять учебные Markdown-файлы. После выполнения работа фиксируется отдельным коммитом и передаётся преподавателю на проверку.

## Требования

- актуальная LTS-версия .NET SDK;
- Git;
- IDE с поддержкой C# (Visual Studio, Rider или VS Code);
- Docker Desktop — начиная с тем о базах данных и интеграции.

Команды в материалах приведены для `dotnet` CLI и одинаково применимы во всех основных ОС.
