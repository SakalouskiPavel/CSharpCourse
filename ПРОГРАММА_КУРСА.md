# Программа курса «Разработчик C#/.NET»

Файл описывает текущую иерархию курса по материалам, расположенным в корне папки. Его можно использовать как основу для добавления теоретических конспектов, расширенных заметок и навигации по темам.

## Общая структура

Курс состоит из 35 занятий и логически делится на следующие блоки:

1. Введение в курс и рабочий процесс.
2. Базовые и объектно-ориентированные возможности C#.
3. Работа с коллекциями, LINQ, строками и регулярными выражениями.
4. Метапрограммирование, сериализация, исключения и работа с данными.
5. Продвинутые возможности языка и новые версии C#.
6. Шаблоны проектирования.
7. Параллелизм, асинхронность и взаимодействие процессов.
8. Архитектура приложений, безопасность, API, кэширование и интеграция.
9. Инженерные практики, CI/CD, брокеры сообщений и методологии разработки.

## Рекомендуемая структура теоретических материалов

Для дальнейшего наполнения курса теорией удобно сохранять порядок существующих занятий и добавлять к каждому разделу однотипные файлы:

| Файл | Назначение |
|------|------------|
| `README.md` | Краткое описание темы, цели занятия и навигация по материалам |
| `theory.md` | Основной теоретический конспект |
| `examples.md` | Разбор примеров и типовых сценариев использования |
| `homework.md` | Описание задания, критерии выполнения и подсказки |
| `links.md` | Дополнительные ссылки и источники |

Если теоретические материалы будут добавляться постепенно, приоритетными для конспектирования выглядят темы 02-17 как языковая база C#, затем темы 21-26 как блок конкурентности и асинхронности, после этого темы 27-34 как архитектурно-прикладной блок.
